# HypercubeLib
3D & GUI rendering library for Minecraft Forge 1.12.2+ mods.

### Main features:
#### 3D & Models:
- Custom model baking
- Model JSON parsing
- Model structure serialization
- Multiple-pass model texturing
- Automatic model rotation

#### GUI:
- Complete GUI event handling
- GUI rendering aids
- GUI rendering helpers
